# Backstage Lab

Lab template and example for Backstage scaffolder, pipelines, and Terraform automation.

## What's in here

- `catalog-info.yaml` - Backstage catalog config
- `templates/` - Scaffolder templates for creating stuff
- `pipelines/` - Example Azure DevOps pipelines
- `terraform/` - Terraform templates

## Before you use this

Replace the placeholders in the template files with your own stuff:

- `<YOUR_AZURE_DEVOPS_SUBDOMAIN>` - your Azure DevOps org
- `<YOUR_ORGANIZATION>` - Azure DevOps org name
- `<YOUR_PROJECT>` - your project name
- `<YOUR_REPO_NAME>` - repo name
- `<YOUR_BRANCH>` - target branch (main, develop, etc)

Just find and replace all the `<...>` with your actual values and you're good to go.
